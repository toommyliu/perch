# perch

Perch is a lightweight macOS menu bar app for viewing your calendar at a glance. It focuses on a simple, read-only experience that keeps your schedule visible without opening a full calendar app.

## Rationale

Notion Calendar is good, but the only reason I use the app is for the menu bar feature.

Perch exists to make that specific workflow faster and more focused: show what is coming up from Apple Calendar directly in the menu bar, with as little overhead as possible.
